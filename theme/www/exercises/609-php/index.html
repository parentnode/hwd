Exercise 609 - Janitor navigation

This exercise is about developing your first Janitor based PHP project

The purpose of this exercise is to work with your personal Janitor project to gain experience with Janitor.

Tasks

  1. Understand navigations

    A Janitor project is born with two default navigation lists. One primary navigation for the frontend (main) and another primary navigation for the backend interface (main-janitor).

    The frontend navigation list is included in your page via the theme/templates/www.footer.php template. There you can also see how to get a navigation list and how you might turn it into HTML.

    You can modify the navigation lists through Janitor. Go to Site -> Navigations and select the navigation list you want to manage. You can also create any number of additional navigation lists, that you can use in your templates.
