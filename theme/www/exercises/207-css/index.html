Exercise 205 - HTML form

This exercise is about styling form HTML.

The purpose of this exercise is to practice writing CSS, by applying a specific interactive "design" to an HTML form. The goal is to make the form html look like the reference graphic, which will be visible when you hover the mouse over the text in the top of the page â€“ without changing the HTML.

Since this is an interactive design, the reference graphic shows the form with the "Last name"-field focused. There is a video in the bottom of this page to show the entire form interaction.

Tasks

  1. Add CSS

    Add your own CSS to the style-tag in the top of the exercise template (205/index.html), to make the form appear exactly as in the reference graphic.

    Applied screenshot is taken in Firefox on Mac. The checkbox is a standard system checkbox, which will look different on different systems.

  2. Validate your CSS on w3c

    Use https://jigsaw.w3.org/css-validator/validator to validate your CSS.

Hints

  • Font-family: Verdana
  • Pseudo classes: :focus, :hover
  • Colors: green, lightgreen, #bbbbbb
  • Think about using the + selector to achive the focus effect
  • A little bit of margin and minus margin for the help div
