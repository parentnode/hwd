Exercise 207 - CSS for forms

This exercise is about styling forms.

The purpose of this exercise is to transfer your KBHFF form css proposal to general form element.

Tasks

  1. Transfer your KBHFF styles

    Transfer and adjust the styles from your KBHFF login page.

  2. Add an indication for required fields

    Some fields are required and has an required class. Find a way to visually indicate if a field is required.

  3. Add styles for error message

    The third text-input has an error class, indicating it has an error. Show the error text for this element.
