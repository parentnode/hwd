Exercise 003 - HTML tables

This exercise is dealing with basic HTML tables (<table>).

There are two purposes of this exercise: To write HTML and to practice deducting the correct semantic structure of an unformatted text.

Tasks

  1. Add correct table HTML to the text in 003/index.html

    The structure of the text needs to be reflected by html. You will need to read the text to identify which table elements to use.

  2. Check your HTML formatting and indenting

    Make sure you opened and closed all tags appropriately and indented all child nodes according to their parents.

  3. Validate your HTML on w3c

    Use https://validator.w3.org/#validate_by_input to validate your HTML.
