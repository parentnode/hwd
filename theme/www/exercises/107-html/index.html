Exercise 107 - HTML structure

This exercise is dealing with basic HTML structuring (<div>).

The purpose of this exercise is to practice writing HTML.

Tasks

  1. Add general page structure

    1. Add a div-tag with the id page, to the empty HTML document in 107/index.html.
    2. Add a div-tag with the id header, to the page-div.
    3. Add a div-tag with the id content, to the page-div.
    4. Add a div-tag with the id navigation, to the page-div.
    5. Add a div-tag with the id footer, to the page-div.

  2. Add content

    Copy the HTML formatted content from Exercise 101, 103, 105 and 106 to the content-div.

  3. Re-prioritise content

    Update all headline tags to fit the hierarchy of the new combined page.

  4. Add structure to content

    Wrap each section, defined by a starting H2 tag, in a new div with the class section.

  5. Check your HTML formatting and indenting

    Make sure you opened and closed all tags appropriately and indented all child nodes according to their parents.

  6. Validate your HTML on w3c

    Use https://validator.w3.org/#validate_by_input to validate your HTML.

  7. Check the document outline

    Use https://gsnedders.html5.org/outliner/ to a get a structural representation of your HTML.
