Exercise 101 - CSS basics

This exercise is about adding basic content layout styling to an HTML document

The purpose of this exercise is to practice basic CSS, by applying a specific "design". The goal is to make the text appear exactly as is looks in the reference graphic, which will be visible when you hover the mouse over the page content â€“ without changing the HTML.

Tasks

  1. Add CSS

    Add your own CSS to the style-tag in the top of the exercise template (101/index.html), to make the text appear exactly as in the reference graphic.

  2. Validate your CSS on w3c

    Use https://jigsaw.w3.org/css-validator/validator to validate your CSS.

Hints

  • Times new roman, Arial
  • red, blue, green, white, black
  • margin, padding, font-family, font-size, font-style, line-height, display, color, border, background
