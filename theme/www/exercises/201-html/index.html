Exercise 201 - Homework TOC

This exercise is about making life a little bit easier and keeping the dots connected.

The purpose of this exercise is to create a small maintenance dependency, that you need to keep in mind onwards.

Tasks

  1. Add a TOC to your homework folder

    Add an index.html file to the root of your personal homework folder, using the 201/index.html as a guideline.

  2. Add links to your exercises

    Add links to every exercise in your personal homework folder.

    Whenever you add a new exercise to your homework folder, you MUST remember to update this list.

  3. Add a link to your homework folder

    In /homework/index.html, add a link to the student list, pointing to the new homework TOC you have just created in your personal homework folder

  4. Validate your HTML on w3c

    Use https://validator.w3.org/#validate_by_input to validate your HTML.
