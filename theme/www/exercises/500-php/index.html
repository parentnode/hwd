Exercise 500 - Introduction to PHP

PHP is a scripting language, in many ways similar to JavaScript. It is very easy to get started with, it is enormously flexible and well covered in both documentation, available libraries and updates and you are about to join a community of more than 6 million PHP developers.

Because now you will learn it too.

PHP scripts have the .php extension. By giving the file the .php extension your webserver knows to execute any php code blocks present in the file, before serving the output to the browser. If you write PHP code inside an .html file, it will come out as text.

A PHP script is only executed in blocks of code between the start tag, <?php, and the end tag, ?>. A shorthand for the start tag, <, can be used in most environments.

PHP can be mixed in with HTML to make the content dynamic. You can add a PHP start tag anywhere in your script, and have regular HTML outside the PHP blocks.

In PHP scripts, you may omit the end script tag, if PHP is the ending block of the script.

Variables are defined using a $-sign, so $string is a variable. Variables does not need to be declared before being used.
