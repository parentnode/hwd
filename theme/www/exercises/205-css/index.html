Exercise 204 - CSS for forms

This exercise is about styling forms.

The purpose of this exercise is to get a first experience with writing CSS for forms. The primary goal is exploration. Seondary goals are listed below. Use google to find more information on styling forms.

Tasks

  1. Add form

    Copy the form and form elements you created in exercise 203 into 204/index.html

  2. Adjust the input width

    Make the form elements as wide as the content div.

  3. Hide the hint- and error-message

    Hide the divs containing hint and error messages as default

  4. Add form layout

    Place the labels above the basic input elements, select and textarea – but next to the checkbox and radiobuttons

  5. Add focus hightlighting

    Change the look of your input elements when they gain focus – AND show the hint message

  6. Add button mouseover effect

    Change the look of your button when you hover the mouse over it

  7. Validate your CSS on w3c

    Use https://jigsaw.w3.org/css-validator/validator to validate your CSS.
