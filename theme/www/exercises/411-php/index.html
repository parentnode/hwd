Exercise 411 - Janitor upgrade

This exercise is about upgrading your Janitor project

The purpose of this exercise is to work with your personal Janitor project to gain experience with Janitor and PHP.

Tasks

  1. Run the install script

    Find the web stack install guide for your platform on parentnode.dk/blog. Run the install script as described in the guide.

    Wait for it to finish and visit your project to verify that the server setup is working as intended.

  2. Pull the latest Janitor

    Open your bash and go to the root of your personal Janitor project. Good. Then continue to submodules/janitor inside your project. Once you are there, run this command to pull the latest Janitor code to your machine:

    git pull origin master

  3. Upgrade Janitor

    Log in to Janitor and go to the backend. Go to System -> Setup via the navigation and then choose Upgrade Janitor from the Maintenance room.

    Janitor will now perform a number of structure and data tests and attempt to update the code where needed.

    The upgrade process must output "Upgrade complete" to have finished correctly.
