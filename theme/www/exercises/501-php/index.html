Exercise 501 - PHP

This exercise is about your first PHP script

The purpose of this exercise is to create a PHP script and execute it on your local webserver to see the output in your browser. As simple as that. A regular old-school hello world example.

Tasks

  1. Create a new PHP file

    In your homework folder, create the 501 exercise folder and create an empty index.php file inside.

  2. Write you first PHP output

    Add a PHP starting tag to you new PHP script (<?php). And then use the php function print to print a string, that will be shown in the browser, when you access the script via the hwd.local website. It works like this:

    print "hello world";

    Feel free to change to text to show you have understood the concept :-)

  3. Check out your script

    View you script output in you browser.

  4. Force an PHP error

    If you got this far without meeting any PHP errors, then this is your turn now. Let's try to force out a PHP error. Try to remove the quotes from the string you print. And run the script again. Now you know what it looks like :-) It contains valuable output that can be used to locate the error.

  5. Use a variable

    Now, let's use a variable to store the string you want to print. So assign the string to a variable and print that variable instead of the string.

Hint

To get the php to run you need to go to your homework/"yourname"/index.html. In here make sure the link ends with php NOT html. like so: <a href="501/index.php">501 php</a>
