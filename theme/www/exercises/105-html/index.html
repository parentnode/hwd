Exercise 203 - HTML form

This exercise is dealing with form HTML.

The purpose of this exercise is to get acquainted with HTML form and input element, by using the templates from Templator.

In the templator snippets some classes are wrapped in brackets []. This means they are optional. You can remove the optional classes in your implementation.

Tasks

  1. Create a form

    Use the templator form snippet to add a form with a fieldset and an actions list to 203/index.html

  2. Add basic inputs

    Use the templator form element snippets to add 3 input types to your form â€“ one for a regular text input, one for an email and one for a password

  3. Add select

    Use the templator form element snippets to add a select element to your form

  4. Add more options to the select

    Figure out how to add more options to your select, by looking at the HTML

  5. Add textarea

    Use the templator form element snippets to add a textarea element to your form

  6. Add checkbox

    Use the templator form element snippets to add a checkbox element to your form

  7. Add radiobuttons

    Use the templator form element snippets to add two radiobutton elements to your form

  8. Add submit button

    Use the templator form element snippets to add a submit button to the actions list in your form

  9. Check your HTML formatting and indenting

    Make sure you opened and closed all tags appropriately and indented all child nodes according to their parents.

  10. Validate your HTML on w3c

    Use https://validator.w3.org/#validate_by_input to validate your HTML.
