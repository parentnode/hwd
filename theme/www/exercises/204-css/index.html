Exercise 202 - CSS for tables

This exercise is about styling tables.

The purpose of this exercise is to practice writing CSS, by applying a specific "design" to a table. The goal is to make the table html look like the reference graphic, which will be visible when you hover the mouse over the page content â€“ without changing the HTML.

Tasks

  1. Add CSS

    Add your own CSS to the style-tag in the top of the exercise template (201/index.html), to make the text appear exactly as in the reference graphic.

    Applied screenshot is taken in Firefox on Mac.

  2. Validate your CSS on w3c

    Use https://jigsaw.w3.org/css-validator/validator to validate your CSS.

Hints

  • Verdana
  • blue, grey, white, black
  • vertical-align, line-height, width, all padding-sizes divisible with 5, no margins
