Exercise 502 - PHP

This exercise is about your first PHP loops and conditionals.

The purpose of this exercise is to use PHP and get familiar with the basics language syntax.

Tasks

  1. Create a new PHP file

    In your homework folder, copy the 502 exercise material folder into your homework folder.

  2. Read through the script

    Read the code inside the work file, 502/index.php. It contains 3 Arrays, one of which is indexed, one that is associative and one that is multidimensional. Notice the different ways values can be added to the array. You will be using for and foreach loops as well as some if statements to complete the exercise.

    Using print_r($indexed); will output the array content in a structured form, although you may need to look at the source output to see it (view page source).

  3. Output $indexed

    Use a for loop to loop through the content of $indexed and print out each value as a <li> in a valid HTML list (<ul>).

    Do not output the "Don't print me" element.

  4. Output $associative

    Use a foreach loop to loop through the content of $associative and print out each key and value as an <dd> and <dt> in a valid HTML definition list (<dl>).

    Do not output the "Don't print me" element.

  5. Output $multidimensional

    Use nested foreach loop to loop through the content of $multidimensional and print out the content in nested html structure, where the indices (the keys of the arrays) becomes headers.
