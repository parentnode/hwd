Exercise 102 - HTML, Your student profile

This exercise is about creating you own student profile for this course. This page will be used throughout the course and allows you to apply what you have learned, in a freestyle manner.

There are two purposes of this exercise: To write HTML and to gain more experience with doing so.

Tasks

  1. Create your profile folder in classroom

    First you create a new folder with your name of choice in the classroom folder. In the classroom folder you will see the folders of other participants. If the name you wanted is already taken, then please select another name for your profile folder.

  2. Copy the clean HTML file to use as a starting point

    In classroom/new-participant you will find an index.html file – copy it to your classroom profile folder to have a starting point for your profile page.

  3. Add some content to your profile page

    With what you have learned in the first exercise, add some html to your profile page. You can add what you want – it is your playground. If you cannot think of anything to add, you can just write something about yourself.

  4. Check your HTML formatting

    Make sure you opened and closed all tags appropriately.

  5. Validate your HTML on w3c

    Use https://validator.w3.org/#validate_by_input to validate your HTML.

  6. Check the document outline

    Use https://gsnedders.html5.org/outliner/ to a get a structural representation of your HTML.
