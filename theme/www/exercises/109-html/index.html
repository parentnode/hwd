Exercise 005 - HTML + SEO

This exercise is about adding schema.org properties to the HTML you created in Exercise 4.

The purpose of this exercise is to practice HTML SEO optimization and working with more complex HTML.

Tasks

  1. Copy the HTML from Exercise 4

    Copy the HTML formatted content of body from Exercise 4 into the index.html of Exercise 5. You need this HTML as the baseline to add schema.org properties to it.

  2. Add schema.org properties to your HTML

    Consider your content as a CreativeWork. Add the relevant properties from the CreativeWork definition on schema.org to your HTML. Don't worry, this is a pretty simple content type.

  3. Check your schema.org properties

    Make sure all your schema.org properties are added correctly to your HTML and that all values are added correctly and wrapped in double-quotes (").

  4. Validate your HTML on w3c

    Use https://validator.w3.org/#validate_by_input to validate your HTML.

  5. Test the structured data

    Use https://search.google.com/structured-data/testing-tool to test the schema.org properties you have added. Fix any errors or warnings and test again.
