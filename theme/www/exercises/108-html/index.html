Exercise 108 - HTML, Your student profile

This exercise is about using what you have learned on your student profile page. You have created this page in exercise 102 and you have learned some things since then. Now it is time for an update.

There are two purposes of this exercise: To write HTML and to gain more experience with doing so.

Tasks

  1. Add structure and content

    Add some structure elements to your profile page. Perhaps some more content. Play with the HTML tags you have learned until now.

  2. Check your HTML formatting

    Make sure you opened and closed all tags appropriately.

  3. Validate your HTML on w3c

    Use https://validator.w3.org/#validate_by_input to validate your HTML.

  4. Check the document outline

    Use https://gsnedders.html5.org/outliner/ to a get a structural representation of your HTML.
