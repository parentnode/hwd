Exercise 102 - CSS specificity

This exercise is about CSS specificity – weight of selectors and combinations of selectors.

The purpose of this exercise is to practice writing targeted CSS. The goal is to make the red box look like the multicolored reference graphic, by adding appropriate CSS selctors to each div, with the right background-color – Without using the !important rule.

Tasks

  1. Add CSS selectors with background-colors

    Add your own CSS selectors to the individual divs, applying the colors shown in the reference graphic.

  2. Validate your CSS on w3c

    Use https://jigsaw.w3.org/css-validator/validator to validate your CSS.
