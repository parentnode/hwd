Exercise 401 - PHP

This exercise is about setting up your first Janitor based PHP project

The purpose of this exercise is to set up a new personal Janitor project on your computer. This project will form the basis for the following PHP exercises.

Tasks

  1. Create a new Git repository

    Log into your Github account and create a new repository. Give it a good name, using only lowercase letters and underscores.

  2. Clone the new repository

    Create a personal folder in /srv/sites/#YOUR NAME# and clone your new repository into this folder. It should say that you cloned an empty repository â€“ this is correct.

  3. Install Janitor

    Open your BASH (as administrator) and cd your way to your new repository. Once you are in the root folder of your repository you can run the install command:

    Mac:

    bash <(curl -s https://anitor.parentnode.dk/installers/submodule-installer-mac)

    Windows:

    bash <(curl -s https://janitor.parentnode.dk/installers/submodule-installer-windows)

    This will start the Janitor install and setup process. You will be prompted for some information along the way, among other things a domain for your local setup. You can choose a name (lowercase letters and dashes) and then add .local, so it ends up something like martins-test.local.

  4. Visit your project

    After the installation and setup, you should be able to see a "Janitor is alive" page when you visit your new project domain.
