Exercise 313 - JS, Manipulator Documentation

This exercise is about cloning and setting up an existing Janitor project.

The purpose of this exercise is to practice setting up new sites on your computer, and to use have the manipulator documentation source code ready to use in later exercises.

Tasks

  1. Clone manipulator_parentnode_dk

    Clone manipulator_parentnode_dk from the parentnode account into sites/parentnode

  2. Enable the site

    Enable the site by using the enable_site bash command or by doing it manually, by modifying the apache.conf and the hosts file.

  3. Visit manipulator.local/janitor/admin/setup

    Follow the setup guide on http://manipulator.local/janitor/admin/setup. You can skip the mail and payment options.

  4. Create a navigation for the site

    Login in on http://manipulator.local/login. Go to the backend via the janitor link in the top right corner of the page (if you are not already there). In the left side there is a "hidden" menu. Under SITE you will find NAVIGATIONS, and here you have a main navigation which is intended for the main frontend navigation.

    Add a new navigation item to the main navigation, using this data set: Name: Docs, Classname: docs, Link: /docs

  5. Test the site

    Visit http://manipulator.local and check that the navigation incudes you new "Docs" item.
