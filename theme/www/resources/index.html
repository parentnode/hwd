How to get the exercises from github

So first of all you should make an account on github

Github is basically a place where you can get access to all the projects that are open to the public, it could be any open source project in the world. but more than just storing a lot of code the essential thing is that its a git system, and git is a way of getting access to older versions of some code localy on your pc. The smart thing about git is if you made a mistake you can go back through all the older versions while it was still working. And on github you can also see what changes there where made by who and lots of other things, you can merge projects fork them ect you can read more about that if you are curious. Well the bottom line is you are gonna upload your projects on to github/parentnode every time you have something worth up loading to save it on a cloud. So if you loose or break your pc you you still have the code and we can see what you are doing. I hope you don't store important things on your pc in general. If you do please get it on a cloud for your own sake. :)
